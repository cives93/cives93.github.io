About CETIS

The most advanced statistical application specifically designed for environmental toxicity testing.

Our Story

Born from a Real Need

Since 1992, Tidepool Scientific Software has been committed to developing and publishing biostatistical software specifically targeted toward the environmental toxicology community.

Prior to the formation of the company, environmental toxicologists were faced with increasing statistical demands and woefully deficient or non-existent software. Initially, a spreadsheet-based application named ToxCalc was developed, which went on to become the best-selling software of its kind with over 500 registered users.

Starting in the year 2000, development efforts for CETIS began in earnest, leveraging the power of the Microsoft Access Relational Database System to create a true database application.

Why CETIS

What Makes CETIS Different

Massive Time Savings

Labs report CETIS requires less than one-tenth the time to input and analyze toxicity data compared to traditional software, representing potential annual savings of $15,000+.

Regulatory Compliance

The only software fully supporting the methodology described in the Guidance Document on Statistical Methods for Environmental Toxicity Tests (Environment Canada, 2005).

Multi-Jurisdictional

Since federal and state programs have different decision-making rules, CETIS provides a system to quickly change decision rules depending on which jurisdiction governs testing.

No SQL Required

You will need to know almost nothing of the Structured Query Language because CETIS does all the work behind the scenes. Focus on science, not database commands.

Comprehensive Statistics

Full suite of advanced statistical methods for hypothesis testing and point estimation, with 41 non-linear regression models and thorough residual analysis.

Data Integrity

MD5 hash digests protect data integrity. Split database architecture ensures secure, multi-user access to shared data repositories.

Versatility

CETIS efficiently stores and processes data from a wide range of toxicity test types and environmental media.

Compatibility

Platform Support

CETIS now runs on Microsoft Access 2010, 2013, 2016, 2019, 2021 and Microsoft 365, supporting both x64 and x86 versions.

The file-server architecture allows the front-end application to be installed on individual workstations while the back-end database resides on a shared server drive, enabling simultaneous multi-user access across the network.

Coming Soon: CETIS Enterprise will support SQL Server and Azure SQL for mission-critical deployments with hundreds of simultaneous users and millions of records.
