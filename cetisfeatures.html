Analysis

Statistical Analysis

A full suite of advanced statistical methods for hypothesis testing and point estimation.

Linear Regression (MLE)

Four models including Log-Cumulative Normal (Probit Analysis), Log-Logit, Log-Gompertz, and Log-Angle. Emulates U.S. EPA Probit software and SAS Probit procedure.

Non-Linear Regression

41 different models including Logistic, Log-Logistic, Cumulative Normal, Gompertz, Weibull, Morgan-Mercer-Flodin, and models with hormesis factors.

Residual Analysis

Thorough residual analysis using Pearson chi-square, likelihood ratio, ANOVA lack of fit, Shapiro-Wilk, Anderson-Darling, Bartlett, Brown-Forsythe Levene, and more.

Compliance

WET Evaluation & Reporting

The CETIS WET Evaluation report is a special report type that assesses a Whole Effluent Toxicity test for compliance with established IWC (Instream Waste Concentration) criteria and test acceptability criteria.

CETIS considers the WET test regulatorily significant if the calculated index of toxicity is less than or equal to the permitted IWC.

New: U.S. EPA Test for Significant Toxicity (TST) using Welch's t Test — a bioequivalence approach that controls the false negative rate.

QC

Quality Control

Three types of quality control plots to monitor your laboratory's reference toxicant performance.

Levey-Jennings

Shewhart-type control charts for monitoring test precision and accuracy over time.

Cumulative Mean

Track the running mean of your reference toxicant data to identify long-term trends.

CUSUM Plot

New two-sided CUSUM plot helps detect subtle trends and shifts in your data over time.

Architecture

Database & Architecture

File-Server Architecture

CETIS uses a split database design — the front-end application runs on each workstation while the back-end database resides on a shared drive. This allows simultaneous multi-user access across your network.

Data Security

MD5 hash digests are calculated and stored for all datasets, ensuring data integrity and providing an audit trail for regulatory compliance.

Productivity Tools

Pop-up calendars, calculators, rapid data entry, Office Ribbon UI, and improved native filtering and exporting capabilities.

Broad Compatibility

Runs on Access 2010, 2013, 2016, 2019, 2021, and Microsoft 365 — both x64 and x86 versions supported.
