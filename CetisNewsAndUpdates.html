Release History

Latest Release

CETIS v2.1.6.xx

The biggest change to the user interface since the introduction of CETIS over 20 years ago. CETIS now implements the Office Ribbon UI with improved native filtering and exporting capabilities. A dramatically redesigned interface increases efficiency and ease of use.

Security Update

CETIS v2.1.3.0 – TLS 1.2 Update

Versions prior to v2.1.3.0 used a version of QLM that required TLS 1.1 for license server communication. As web hosting services abandoned TLS 1.0 and 1.1, CETIS was rewritten with TLS 1.2–capable QLM v16, incorporating a more capable Soraco-designed wizard for both online and offline activation or deactivation.

Users are advised to update as soon as possible. Free updates for purchases less than one year old.

Major Release

CETIS v2.1 – 64-bit Support

You asked for 64-bit compatible CETIS and now you got it! Version 2.1 introduced full 64-bit compatibility alongside the newly designed user interface that dramatically increases efficiency and ease of use. CETIS now runs on Access 2010, 2013, 2016, 2019, 2021, and Microsoft 365.

Enhancement

EPA TST Method Added

Added the new U.S. EPA Test for Significant Toxicity (TST) using Welch's t Test — a bioequivalence approach that controls the false negative rate for more reliable toxicity determinations.

Enhancement

Ratio Tests & Comparison Methods

New implementation of the Robertson and Preisler ratio test and the Zajdlik two-sample Z test for comparing two different point estimates.

Enhancement

Data Security Improvements

New data security measures implemented including the calculation and storage of MD5 hash digests of datasets for data integrity verification.

Enhancement

CUSUM Quality Control Plot

A new two-sided CUSUM plot was added to the quality control toolkit, helping labs detect subtle trends in their reference toxicant data.

Support

Self-Help Portal

A standalone QLM Wizard and Self-Help Portal were added to the website under the Support menu for legacy users needing license management assistance.
