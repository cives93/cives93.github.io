Overview

What is CETIS?

A Microsoft Access–based application built by environmental toxicologists, for environmental toxicologists.

Data Analysis

Full suite of advanced statistical methods for hypothesis testing and point estimation, including probit analysis, non-linear regression with 41 models, and EPA-approved methods.

Database Management

Efficiently stores and processes data from freshwater, estuarine, marine, and terrestrial toxicity tests on effluents, ambient water, reference toxicants, and sediment samples.

Decision-Tree Automation

Automated analyses using customizable decision-tree logic consistent with U.S. EPA, Environment Canada, and other regulatory authorities. Override any decision manually at will.

Architecture

Built for Multi-User Environments

CETIS uses a file-server architecture with split front-end/back-end databases, allowing simultaneous multi-user access.

Front-End

Installed on each workstation, providing the user interface and analytical tools.

Back-End Database

Installed on a shared server drive, serving as the main data repository for all users.

Data Security

MD5 hash digests ensure data integrity and security across all stored datasets.
